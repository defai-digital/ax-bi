# ax-bi

![Version: 0.17.3](https://img.shields.io/badge/Version-0.17.3-informational?style=flat-square)

AX BI is a GenAI-native business intelligence platform

**Homepage:** <https://github.com/defai-digital/ax-bi>

## Source Code

* <https://github.com/defai-digital/ax-bi>

## TL;DR

```console
helm dependency build helm/ax-bi
helm install ax-bi helm/ax-bi -f helm/ax-bi/values-axbi.yaml
```

Make sure you set your own `SECRET_KEY` to something unique and secret. This secret key is used by Flask for
securely signing the session cookie and will be used to encrypt sensitive data in AX BI's metadata database.
It should be a long random bytes or str.

On helm this can be set on `extraSecretEnv.AX_BI_SECRET_KEY` or `configOverrides.secrets`

The AX BI values file deploys the web app, Celery worker, Celery beat,
MCP service, and `ax-services` sidecar. For production, pin image tags,
replace all placeholder secrets, use managed Postgres/Redis where possible,
and enable JWT authentication for MCP.
